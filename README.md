# Password Security Analyzer

A Python cybersecurity tool with 3 features.

## Features
- Password Strength Checker
- HaveIBeenPwned Breach Detector
- Random Password Generator

## Technologies Used
- Python
- hashlib (SHA-1 hashing)
- urllib (API connection)

## What I Learned
- SHA-1 hashing and k-anonymity
- API integration
- Cybersecurity best practices
